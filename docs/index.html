Features

Not just a pretty diagram

Content intelligence that surfaces what's interesting — contradictions, gaps, consensus, and hidden connections.

⚡

Contradiction Detection

When sources disagree on a claim, both nodes get flagged and connected with a red dashed arc. You see the tension, not a false consensus.

❓

Gap Analysis

Structurally important areas with thin evidence get inquiry nodes — dashed borders that signal "this is a question, not an assertion."

⤸

Cross-Branch Links

Dotted arcs connect ideas across branches when they have non-obvious relationships. Labeled with verbs like "enables" or "blocks."

◎

Focus Mode

Click any branch to isolate it — everything else fades to 8%. Navigate dense maps without needing separate pages.

⊕

Progressive Disclosure

Toggle between keyword-only map view and full detail reading mode. One artifact, two uses.

🔗

Source Attribution

Nodes traced to their source with hover links and a footer reference list. No fabricated citations — ever.

🎨

4 Color Palettes

Switch between Bauhaus, Ocean Sunset, Nordic Forest, and Pastel Garden at runtime. Exports use the active palette.

↓

6 Export Formats

SVG, PNG (2× retina), PDF (native vector), Markdown, and Mermaid. Portable font stack that renders everywhere.

🧲

Semantic Gravity

Related branches sit adjacent on the circle. Physical proximity encodes conceptual relatedness — space becomes meaning.

Each palette is designed for a different tone. Switch at runtime — the map recolors instantly.
