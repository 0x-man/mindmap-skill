Features

Not just a pretty diagram

Content intelligence that surfaces contradictions, gaps, consensus, and hidden connections.

⚡

Contradiction Detection

When sources disagree, both nodes get flagged and connected with a red dashed arc.

❓

Gap Analysis

Thin evidence areas get inquiry nodes — dashed borders signaling questions, not assertions.

⤸

Cross-Branch Links

Dotted arcs connect ideas across branches with relationship labels like "enables" or "blocks."

◎

Focus Mode

Isolate any branch — everything else fades to 8%. Navigate dense maps effortlessly.

⊕

Progressive Disclosure

Toggle between keyword-only map view and full detail reading mode.

🔗

Source Attribution

Nodes traced to their origin with hover links and a footer reference list.

🎨

4 Color Palettes

Bauhaus, Ocean Sunset, Nordic Forest, Pastel Garden — switchable at runtime.

↓

6 Export Formats

SVG, PNG (retina), PDF (vector), Markdown, and Mermaid.js.

🧲

Semantic Gravity

Related branches sit adjacent. Physical proximity encodes conceptual relatedness.
